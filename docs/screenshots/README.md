## How to Take Screenshots

### Windows
- **Snipping Tool**: `Win + Shift + S`
- **Full Screen**: `PrtScn` key
- **Active Window**: `Alt + PrtScn`

### Recommended Settings
- **Resolution**: 1920x1080 or higher
- **Format**: PNG (lossless)
- **Browser**: Chrome (for consistent rendering)
